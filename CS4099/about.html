This is a Mockup of a news webpage to show the ability of an IFrame exploration script. The script looks to expose the source of an IFrame object, by clicking on any of the advert IFrames the source should replace the advert image. This script is also encorporated into a Plugin information about which can be found here. This plugin was created in cooperation with an industry contact, who had requested a simple way for employees to identify the origin of adverts.
